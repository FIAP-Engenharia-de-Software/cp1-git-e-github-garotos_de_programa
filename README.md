# Case-Social-Connect
Repositório que servirá como 1º Checkpoint da disciplina de WebDev da FIAP


**Resumo do projeto** 

O SocialConnect é uma plataforma de rede social destinada a conectar profissionais e empresas. O projeto envolve a criação e manutenção de um site com várias seções, incluindo informações sobre a empresa, equipe, contatos e links para redes sociais. O desenvolvimento é feito de forma colaborativa, utilizando Git e GitHub para versionamento e gerenciamento do código, além de implementar as melhores práticas de desenvolvimento.

**Tecnologias Utilizadas**
HTML: Estruturação do conteúdo da página.
CSS: Estilização das páginas.
GitHub: Hospedagem do código e uso de GitHub Pages para a publicação.
